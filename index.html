Colors

The colors shown in this system were selected from a split-complements color scheme.

Name Value Usage
Off-White color: hsl(135, 0%, 95%) Used to format the background of the web page (outside of dark mode styles).
Spring Green color: hsl(135, 90%, 80%) As one of the two primary colors used to stylize the web page.
Future Blue color: hsl(200, 90%, 80%) As one of the two primary colors used to stylize the web page. Used in place of Spring Green when two nearby elements need to stand out from each other.
Vibrant Red color: hsl(350, 90%, 80%) As bold contrast to the other colors, used to grab the user's attention and highlight important elements on the page.
